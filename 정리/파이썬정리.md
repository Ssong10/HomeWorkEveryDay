```python

'Hello, %s' % name  # %-formatting
```

### float 값 조심해야할 것

```python
a = 3.5 - 3.12  # 0.37999999999
b = 0.38

abs(a-b) <=1e-10   # 확인법 1

import sys          # 확인법 2
abs(a - b) <= sys.float_info.epsilon

import math            # 확인법 3
math.isclose(a,b)
```
